# Census Analytics Pro 📊

> Granular socio-economic profiling, forecasting, and visualization for Indian Census data.

**Census Analytics Pro** is a comprehensive full-stack web application designed to analyze, visualize, and forecast demographic and socio-economic data across India. By leveraging historical census data and modern machine learning techniques, it provides actionable insights at the national, state, and district levels.

## 🌟 Key Features

*   **Interactive Dashboard**: Real-time overview of national metrics including total population, literacy rates, sex ratio, and live birth/death estimations.
*   **District Deep-Dive**: Granular profiling for any Indian district, covering demographics, socio-economic status, and infrastructure metrics.
*   **Trends & Forecasting**: ML-driven decadal analysis with 21st-century projections (e.g., population growth curves out to 2050).
*   **Geospatial & State Comparisons**: Compare performance across different regions seamlessly.
*   **Export Capabilities**: Download reports instantly in CSV format.

## 🛠️ Technology Stack

**Frontend:**
*   [React](https://reactjs.org/) (Vite)
*   [Tailwind CSS](https://tailwindcss.com/) for styling
*   [Tremor](https://www.tremor.so/) for beautiful charts and components
*   [Lucide React](https://lucide.dev/) for icons

**Backend:**
*   [Python](https://www.python.org/) 3.x
*   [FastAPI](https://fastapi.tiangolo.com/) for high-performance API endpoints
*   [Pandas](https://pandas.pydata.org/) & [Scikit-learn](https://scikit-learn.org/) for data processing and forecasting

## 🚀 Getting Started

### Prerequisites
*   Node.js (v16+)
*   Python (3.8+)

### 1. Backend Setup
Navigate to the backend directory, install dependencies, and run the server:
```bash
cd backend
pip install -r requirements.txt
python main.py
```
*The API will be available at `http://localhost:8000`*

### 2. Frontend Setup
Navigate to the frontend directory, install dependencies, and start the development server:
```bash
cd frontend
npm install
npm run dev
```
*The web app will be available at `http://localhost:5173` (or the port specified by Vite)*
